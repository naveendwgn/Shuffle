## Technologies

Shuffle is built using the following technologies:

- Node.js
- Vite
- React
- Express
- MongoDB
- JavaScript
- HTML
- CSS
- Tailwind  CSS
- DALL-E API
## Installation

To run this project, install it locally using npm:

```
$ clone this repo
$ npm install
$ npm run dev
```
Also make sure you run the backend server from [here](https://github.com/naveendwgn/Shuffle-Server)
